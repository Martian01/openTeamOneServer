Open Team One

This is a first and unrefined version of a web app for Open Team One. It requires Javascript in the browser.

It would be great if we could have a modern and pretty web app at some point, with smooth CSS animations and the like. This is definitely an area where experienced web developers can contribute. ^_^

Open Team One supports multiple styles and skins so that competing web app designs can co-exist.
