Open Team One Administration

This is a first and unrefined version of a web app for server administration. It requires Javascript in the browser.

It would be great if we could have a modern and pretty web app at some point, with smooth CSS animations and the like. This is definitely an area where experienced web developers can contribute. ^_^

There are already measures in place to support multiple styles or skins so that competing designs can co-exist.

Watch this space!
